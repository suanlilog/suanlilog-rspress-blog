# 星露谷下载Mod教程

## 注册并且登录N网账号
[Stardew Valley Mods - Nexus Mods](https://www.nexusmods.com/games/stardewvalley)
（首次安装mod前需要在steam打开星露谷一次）

## 在N网下载SMAPI（Mod启动器）
[SMAPI - Stardew Modding API at Stardew Valley Nexus - Mods and community](https://www.nexusmods.com/stardewvalley/mods/2400)

解压SMAPI，exe安装！！

到steam右键星露谷属性

找到启动选项

把这里全部复制进去，从开始的"双引号"一直到最后的%百分号%
![b4ab54a8171f90831511f8b7d7d729e.png](https://obsidian-hexo-blog-1357454370.cos.ap-beijing.myqcloud.com/hexo/b4ab54a8171f90831511f8b7d7d729e.png)

## 下载并安装Mod
找到你想要的Mod，依然是`FILES->Manual download->Slow download`
**有些Mod有前置Mod要求，会写在弹出的提醒中，也要下载！！**

把Mod解压

在Steam打开"浏览本地文件"
![image.png](https://obsidian-hexo-blog-1357454370.cos.ap-beijing.myqcloud.com/hexo/20260327181046917.png)

复制粘贴解压好的Mod，到`Mods`文件夹中
![1774606357084.png](https://obsidian-hexo-blog-1357454370.cos.ap-beijing.myqcloud.com/hexo/1774606357084.png)
这个文件夹中可以嵌套文件夹，便于自己为Mod分类

## 打开游戏！
直接从Steam中打开星露谷
**会弹出一个黑色框框，游戏过程中千万不要关掉！**
