## 安装插件oh my opencode
项目地址：[code-yeongyu/oh-my-openagent: omo; the best agent harness - previously oh-my-opencode](https://github.com/code-yeongyu/oh-my-openagent)
直接看中文文档：[oh-my-openagent/README.zh-cn.md at dev · code-yeongyu/oh-my-openagent](https://github.com/code-yeongyu/oh-my-openagent/blob/dev/README.zh-cn.md)
![image.png](https://obsidian-hexo-blog-1357454370.cos.ap-beijing.myqcloud.com/hexo/20260327162711739.png)

按照README说的，直接把这些提示词扔到opencode聊天框，然后opencode会问你一系列API配置的问题（Claude，GPT，Gemini，GLM，KIMI等）。

要是你拥有其中一种模型的API，填上去，基本上就完成了安装。
